\# Module Extraction AI Agent



\## Description

Extracts modules and submodules from documentation websites.



\## Installation

pip install -r requirements.txt



\## Usage

python module\_extractor.py --url https://wordpress.org/documentation/



\## Assumptions

\- h2 tags → modules

\- h3 tags → submodules



\## Limitations

\- No deep crawling

\- HTML docs only
